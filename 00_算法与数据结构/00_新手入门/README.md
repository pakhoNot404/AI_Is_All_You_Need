## 位运算



- int：32位有符号整型，0~30位表示数值，最高位表示正负号，所以是-2<sup>31</sup>~2<sup>31</sup>-1;1表示是负数，值是后面的数取反在加1；
- 左移后之前位置补0
- 任何数值，左移一位就相当于这个数×2
- 与1进行与操作，可以判断这个数是0还是1

```java
// 下面代码能够将十进制数转换为2进制数
for (int i = 31; i >= 0; i--) {
            System.out.print((num & (1 << i)) == 0 ? "0" : "1");
}
```
